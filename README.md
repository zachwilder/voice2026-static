# VOICE 2026 - Conference Website

Website for VOICE.advantest.com - The Advantest Developer Conference for semiconductor test professionals.

**Conference:** May 11-13, 2026 | Austin, Texas

## Quick Start

### Development Server (Recommended)

Start the local development server with live reload:

```bash
./start-dev.sh
```

This will:
- Start a server at http://localhost:8000
- Automatically open your browser
- Watch for changes and reload automatically

### Manual Viewing

Alternatively, open `static_mockup/index.html` directly in your browser.
